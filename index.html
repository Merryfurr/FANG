Welcome to FANG

FANG is an open source Python based development language that is meant to be easy. Mostly meant for people who want to start coding but don't want to start a huge development language that has 1000 paged book on it. Cough Cough Python Cough Cough

Get started

Well first you will need to download it! Click here to download a FANG stable release, then head over to the FANG Offical Wiki to see how to make your first program called "Hello World" and then export it!
